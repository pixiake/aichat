A [Flarum](http://flarum.org) extension. demo

## Installation

Install with composer:

```sh
composer require pixiake/aichat:"*"
```

## Updating

```sh
composer update pixiake/aichat:"*"
php flarum migrate
php flarum cache:clear
```
